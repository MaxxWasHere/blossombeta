# Changelog

All notable **beta** releases are documented here. Stable history is on [Blossom](https://github.com/MaxxWasHere/Blossom/releases).

## [2.1.0-beta.1] — 2026-06-01

### Beta program
- Discord **`/getkey`** license keys (one key per user, HWID-bound)
- Online activation against the Blossom license server
- Build expiry date enforced for beta channel
- **`/downloadbeta`** in Discord for direct download (beta role)

### Automations
- Auto Merchant rewritten — single **E** interact, OCR-based shop flow
- Auto daily quests with smarter scheduling vs merchant/inventory tasks
- BR / SC and glitched buff handling improvements
- Task scheduler avoids automations fighting over open UI

### UI
- Blossom 2.0 visual revamp (compact layout, motion, themes)
- Multi-slide intro / setup (webhooks, Roblox, automations)
- Floating side panels (stats, quick settings)
- Always-on-top and session status in sidebar

### Discord webhooks
- Compact biome embeds (Cyberspace, Dreamspace, Glitched highlights)
- Currency screenshot on interval / on start
- Merchant and macro lifecycle notifications
- Per-biome toggles, colors, thumbnails, pings

### Potions & crafting
- Drag-to-mark calibration, Auto Craft controls
- Improved craft timing and safety checks

### Fixes
- Webhook spam / oversized biome images reduced
- Layout fixes (horizontal controls no longer stack incorrectly)
- Merchant UI detection and interaction stability
